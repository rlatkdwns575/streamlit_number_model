# 프로그래밍 학습 AI 챗봇

Streamlit 기반의 역할 전환형 AI 챗봇입니다.  
사용자는 원하는 역할을 선택한 뒤 질문을 입력하면, 해당 역할에 맞는 방식으로 답변을 받을 수 있습니다.

이 프로젝트는 단순 질의응답이 아니라, **역할(Role)에 따라 답변 스타일과 설명 방식이 달라지는 학습용 챗봇**을 목표로 합니다.

---

## 프로젝트 소개

이 앱은 OpenAI API와 Streamlit을 활용하여 만든 **프로그래밍 학습 보조 챗봇**입니다.

사용자는 사이드바에서 역할을 선택할 수 있으며, 현재 구현된 역할은 다음과 같습니다.

- 파이썬 튜터
- 딥러닝 멘토
- 코드 리뷰어

각 역할은 서로 다른 시스템 프롬프트를 가지며, 같은 질문을 하더라도 설명 방식과 답변 형식이 달라집니다.  
예를 들어, 파이썬 튜터는 코드 예시 중심으로 설명하고, 딥러닝 멘토는 비유와 개념 설명에 더 집중하며, 코드 리뷰어는 문제점 분석과 개선 코드 제시에 초점을 둡니다.

---

## 주요 기능

### 1. 역할 전환형 대화
사용자는 사이드바에서 원하는 역할을 선택할 수 있습니다.

- **파이썬 튜터**: 초보자를 위한 기초 설명 + 실행 가능한 코드 예시
- **딥러닝 멘토**: CNN, RNN, YOLO 등 딥러닝 개념을 비유와 함께 설명
- **코드 리뷰어**: 코드의 문제점을 짚고 개선 방향과 수정 코드를 제안

### 2. 실시간 스트리밍 응답
OpenAI API 호출 시 `stream=True`를 사용하여, 답변이 한 번에 출력되는 것이 아니라 실시간으로 생성되는 과정을 바로 확인할 수 있습니다.

### 3. 대화 히스토리 유지
`st.session_state["messages"]`를 활용하여 이전 대화 내용을 유지합니다.  
이 덕분에 사용자는 이전 맥락을 이어서 자연스럽게 대화를 진행할 수 있습니다.

### 4. 역할 변경 시 대화 초기화
역할이 바뀌면 기존 대화 기록을 자동으로 초기화하여, 새로운 역할에 맞는 일관된 대화를 시작할 수 있습니다.

### 5. 수동 대화 초기화
사이드바의 **대화 초기화** 버튼을 통해 현재 대화를 직접 초기화할 수 있습니다.

### 6. API 키 검증
앱 실행 시 `.env` 파일에서 `OPENAI_API_KEY`를 불러오며, 키가 없으면 에러 메시지를 출력하고 실행을 중단합니다.
