# MNIST 숫자 구별 모델

Streamlit과 PyTorch를 이용하여 만든 **숫자 이미지 분류 웹 애플리케이션**입니다.  
사용자가 숫자가 적힌 이미지를 업로드하면, 학습된 CNN 모델이 **0~9 숫자 중 무엇인지 예측**하고, 각 숫자에 대한 확률도 함께 보여줍니다.

---

## 1. 프로젝트 개요

이 프로젝트는 **MNIST 데이터셋**으로 학습된 CNN(합성곱 신경망) 모델을 사용하여  
업로드된 숫자 이미지를 판별하는 웹 앱입니다.

주요 기능은 다음과 같습니다.

- 숫자 이미지 업로드
- MNIST 형식에 맞는 자동 전처리
- 예측 숫자 출력
- 예측 확률(확신도) 출력
- 0~9 각 숫자별 확률 시각화

---

## 2. 사용 기술

- **Python**
- **PyTorch**
- **Torchvision**
- **Streamlit**
- **Pillow(PIL)**

---

## 3. 모델 구조

본 프로젝트는 다음과 같은 CNN 구조를 사용합니다.

- Convolution Layer 1: `1 → 32`
- Convolution Layer 2: `32 → 64`
- Max Pooling
- Dropout
- Fully Connected Layer 1: `9216 → 128`
- Fully Connected Layer 2: `128 → 10`
- 출력: `0~9` 숫자 클래스에 대한 로그 확률

### 모델 클래스
- 클래스명: `CNN`
- 출력 함수: `log_softmax`

---

## 4. 입력 이미지 처리 방식

업로드된 이미지는 MNIST 모델 입력 형식에 맞게 다음과 같이 전처리됩니다.

1. **RGB 이미지 로드**
2. **색상 반전**
   - 흰 배경에 검은 숫자 형태를 MNIST와 유사하게 맞추기 위함
3. **흑백 변환**
4. **28×28 크기 조정**
5. **Tensor 변환**
6. **배치 차원 추가**
   - 최종 입력 shape: `(1, 1, 28, 28)`

---

## 5. 실행 화면 기능

앱을 실행하면 다음 과정을 수행할 수 있습니다.

### 1) 이미지 업로드
- `png`, `jpg`, `jpeg` 형식 지원

### 2) 원본 이미지 확인
- 사용자가 업로드한 이미지를 그대로 보여줌

### 3) 전처리 결과 확인
- 모델 입력용으로 변환된 28x28 흑백 이미지를 시각적으로 제공

### 4) 예측 결과 출력
- 예측된 숫자 출력
- 해당 숫자에 대한 확신도 표시

### 5) 확률 분포 시각화
- 0~9 숫자 각각에 대한 예측 확률을 막대 그래프로 제공
