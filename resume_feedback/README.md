# 자소서 첨삭 AI

Streamlit 기반의 자기소개서 첨삭 웹 애플리케이션입니다.  
사용자가 지원 직무와 자기소개서 내용을 입력하면, OpenAI API를 활용해 강점, 개선점, 수정된 자소서를 생성합니다. 앱은 직무 선택, 첨삭 결과 표시, 결과 다운로드 기능을 포함하고 있습니다. 

---

## 프로젝트 소개

이 프로젝트는 취업 준비생이 작성한 자기소개서를 더 구체적이고 설득력 있게 다듬을 수 있도록 돕기 위해 제작되었습니다.

사용자는 지원 직무를 선택하고 자기소개서 초안을 입력한 뒤, 버튼을 클릭해 AI 첨삭 결과를 받을 수 있습니다.  
출력 결과는 다음 형식으로 구성됩니다.

- 강점 3가지
- 개선점 3가지
- 수정된 자소서 완성본

또한 원문에 없는 허위 경험이나 성과를 임의로 만들어내지 않도록 프롬프트에 제약을 두어, 보다 현실적인 첨삭 결과를 제공하도록 구성되어 있습니다.

---

## 주요 기능

### 1. 직무 선택
사용자는 다음 직무 중 하나를 선택할 수 있습니다.

- Data Scientist
- Software Engineer
- Data Analyst
- Machine Learning Engineer

현재 코드에는 마케팅과 개발 직무에 대한 few-shot 예시가 포함되어 있으며, 이를 기반으로 첨삭 스타일을 유도하도록 되어 있습니다. 

### 2. 자기소개서 입력
텍스트 박스를 통해 첨삭받을 자기소개서를 입력할 수 있습니다.  
입력한 글자 수가 실시간으로 표시되며, 20자 미만일 경우 첨삭이 진행되지 않도록 제한되어 있습니다.

### 3. AI 첨삭 결과 생성
OpenAI API를 사용해 자기소개서를 분석하고 다음 순서로 결과를 생성합니다.

1. 강점
2. 개선점
3. 수정된 자소서

프롬프트는 격려하되 솔직한 멘토 어조를 유지하도록 설계되어 있습니다.

### 4. 결과 다운로드
생성된 첨삭 결과는 `.txt` 파일로 다운로드할 수 있습니다.

## 사용 기술

- Python
- Streamlit
- OpenAI API
- python-dotenv

업로드된 의존성 파일에는 `streamlit`, `openai`, `dotenv`가 포함되어 있습니다.
